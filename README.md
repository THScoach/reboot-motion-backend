# Catching Barrels PWA

Progressive Web App for baseball swing analysis with Live Mode and KRS scoring.

## Tech Stack

**Frontend:**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand (state management)
- TensorFlow.js + MediaPipe (pose detection)

**Backend:**
- Python + FastAPI
- Supabase (Postgres + Auth + Storage)

**Deployment:**
- Vercel (frontend)
- Railway (backend)

## Project Structure

```
/app                    # Next.js frontend
/backend               # Python analysis server
/supabase             # Database migrations
/docs                 # Documentation
```
